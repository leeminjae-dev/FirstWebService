# README
이 레포는 이동욱님의 "스프링 부트와 AWS로 혼자 구현하는 웹 서비스" 책을 읽고 스프링 부트를 활용해서 웹서비스를 만들며 연습해보는 레포입니다.

## 게시판 웹어플리케이션 제작
CRUD 기능이 있는 게시판 웹 어플리케이션을 제작해 봄으로써 스프링을 숙달하고, 데이터베이스를 사용하기 위한 기초적인 4가지
쿼리 형식을 배워본다.
### 기능 구현 목록
- 게시판 기능
    - 게시글 조회
    - 게시글 등록
    - 게시글 수정
    - 게시글 삭제
- 회원기능
    - 구글/네이버 로그인
    - 로그인한 사용자 글 작성 권한
    - 본인 작성 글에 대한 권한 관리
